Providing New Feature

Feature Represent a new HLD ,businsess request or test cases

Clone or Update the repository to its latest


Start Creating a new feature branch from master branch

git checkout master
git checkout -b MYFEATURE

Now Add the feature changes to your feature branch and commit
git add . --all
git commit -m "message"
And merge the feature to develop
git checkout develop
git merge MYFEATURE

Then publish it to gitlab and delete the feature branch
git push -u origin develop
git branch -d MYFEATRE

Go to project develop branch ,Create a merge request between develop and master

To get the published feature

If you don't have the repository locally ,cloning it will give you the latest code


Update your local repository

git checkout develop
git pull

Continue to Maintian Gitlab updated
